#### 项目使用

``` bash
# 克隆项目
git clone https://github.com/MetaEth/codetrade-admin.git

# 进入项目目录
cd codetrade-admin

# 安装依赖
npm install

# 启动服务
npm run dev

# 构建测试环境
npm run build:stage

# 构建生产环境
npm run build:prod

# 建议不要直接使用 cnpm 安装依赖，会有各种诡异的 bug。可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

```
